# 商品属性文案位置

后续如果你要自己调整爱心商户端“商品属性”页面里的字段文案，优先改这里：

- 页面文件：`apps/mobile/src/pages/merchant/templates.vue`
- 页面标题：`商品属性`
- 主副标题：
  - `爱心商户商品属性`
  - `维护和 PC 端一致的货品基础信息，补货时直接复用。`
- 字段文案：
  - `商品编号`
  - `商品全称`
  - `商品名称`
  - `商品大类`
  - `分类`
  - `包装形式`
  - `商品规格`
  - `厂家`
  - `默认数量`
  - `默认保质天数`
  - `图片（选填）`
- 按钮文案：
  - `新增商品属性`
  - `保存修改`
  - `选择拍摄或本地图片`
  - `取消编辑`
- 分类下拉选项：
  - `食品`
  - `饮品`
  - `日用品`

如果你要改管理员或普通用户端的文案，建议直接在对应页面文件里搜索中文原文再改。当前这一页没有单独再抽一层文案常量，目的是让你后续定位更直接。
