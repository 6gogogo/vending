# 文档目录

本目录用于补充“怎么运行、怎么测试、代码结构怎么看、需要掌握哪些知识点”这类说明。

## 文件说明

- `运行与测试说明.md`：后端、后台、移动端、微信小程序的运行与测试方法。
- `完整本地联调流程.md`：从虚拟柜机、移动端开柜到后台监控的一整套本地闭环操作步骤。 
- `sandbox命令清单.md`：所有沙箱调试命令、作用、请求目标和载荷文件说明。
- `Ubuntu环境配置.md`：Ubuntu 服务器或开发机上的环境准备、安装与运行步骤。
- `knowledge/admin-web.md`：电脑端后台的目录结构与知识点。
- `knowledge/mobile.md`：移动端的目录结构、运行方式与知识点。
- `knowledge/api.md`：后端接口服务的目录结构与知识点。
- `knowledge/shared-and-tooling.md`：共享包、工作区脚本、构建和类型检查说明。
