## 你自己添加时怎么放

原则：
- 菜单、tab、`pages.json` 里直接写路径的图，优先放到 `apps/mobile/src/static`
- PC 页面里需要 `import` 的图，优先放到 `apps/admin-web/src/assets`
- 文件名用英文、小写、短横线
- 不要用空格和中文文件名

例子：
- 设置菜单图标：`apps/mobile/src/static/tabs/settings.png`
- 普通页面小图标：`apps/mobile/src/static/icons/nearby-device.png`
- PC 后台菜单图标：`apps/admin-web/src/assets/icons/device-monitor.svg`
- PC 页面插图：`apps/admin-web/src/assets/images/empty-state.png`

## 你添加完后怎么引用

移动端 tab 图标：
- 直接在 [apps/mobile/src/pages.json](/F:/Data/work%20document/codex/vending%20machine/apps/mobile/src/pages.json) 里改 `iconPath` 和 `selectedIconPath`

移动端普通页面图片：
- 直接写相对路径，例如：
```vue
<image src="/static/icons/settings.png" mode="aspectFit" />
```

PC 页面图片：
- 在 Vue 文件里引入，例如：
```ts
import settingsIcon from "../assets/icons/settings.svg";
```
